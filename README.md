## ⚡ Developer Setup Guide

Ensure you have [Node.js](https://nodejs.org) (v18+) installed before continuing.

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Local Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser. The workspace auto-updates dynamically as you perform edits.

### 3. Compile Production Bundle
To verify build stability and type safety:
```bash
npm run build
```

## 🔬 Core System Algorithms

### Geocoded Decay Model
Unlike standard regional targeting, onu.ai implements a geocoded **Haversine Distance Decay algorithm** to model cultural and purchasing dilution:
*   **Optimal Lock (10 KM)**: Demographic purity is maximum (98% signal confidence). Personal dialect matching remains extremely strict.
*   **Wide Sweep (>10 KM)**: Purity decays at `radius * 0.46` BDT weight. The simulated engine models geographical drift, showcasing dialect fusion and transactional habit changes in real time.

### Multimodal Swarm Scan
Campaign creatives undergo parallel processing:
1.  **Vocal Track (Audio)**: Analyzes phonetic speed and tone registers.
2.  **Visual Track (OCR)**: Scans texts and graphic assets for merchant conversion hooks.
3.  **NLP Track (Social Copy)**: Rewrites titles and captions to incorporate regional landmark anchors and culturally high-resonance idioms.
